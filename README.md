# 深度学习课程实验

中国科学院大学 人工智能学院 2025春季学期深度学习课程作业

## 实验列表

| 编号 | 主题 | 模型/方法 | 数据集 |
|------|------|-----------|--------|
| Exp1 | 图像分类 | ResNet | CIFAR-10 |
| Exp2 | 图像分类 | Vision Transformer | CIFAR-10 |
| Exp3 | 文本生成 | 字符级RNN语言模型 | 唐诗 |
| Exp4 | 机器翻译 | Transformer | Multi30k (En→De) |
| Exp5 | 目标检测 | YOLOv5 | 自定义数据集 |
| Exp6 | 语义分割 | SegNet | CamVid |
| Exp7 | 语言建模 | LSTM | Penn Treebank |
| Exp8 | 图像描述 | CNN-Transformer | Flickr8k |
